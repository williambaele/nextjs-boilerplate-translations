# Basic NextJS boilerplate with next-intl & TailwindCSS

This project has:
* Locales for languages (EN/FR)
* Dynamic paths [locales]
* Dynamic content for SEO
    * Title
    * Metadescription
    * Html lang
* Path translations, e.g:
    * /about (EN)
    * /a-propos (FR)


1) Clone
```
git clone <link>
```
2) Install packages
```
npm i
```
3) Run the project
```
npm run dev
```
